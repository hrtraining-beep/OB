# Nose Tea LIFF Register

Deploy this folder to Cloudflare Pages.

After deployment, set the LIFF Endpoint URL to the Cloudflare Pages URL, for example:

```text
https://your-project.pages.dev/
```

The page posts registration data to the Cloudflare Worker backend.

Before deploying, replace `REPLACE_WITH_WORKER_SUBDOMAIN` in `index.html` with your Worker URL.
